# MateMatch 微信小程序使用指南

## 项目概述

MateMatch 是一个"轻招募搭子"微信小程序，帮助用户找到临时的志同道合伙伴进行各类活动，如一起看展、AA聚餐、运动健身等，无需麻烦熟人朋友。

## 快速开始

### 前置要求
1. 微信开发者工具 (下载地址: https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 微信开发者账号 (或使用测试账号)

### 安装步骤

1. **下载项目代码**
   ```bash
   git clone https://github.com/Avel-Z/MateMatch.git
   cd MateMatch
   ```

2. **打开微信开发者工具**
   - 点击"导入项目"
   - 选择项目目录
   - 输入 AppID (或选择"测试号")
   - 项目名称: MateMatch
   - 点击"导入"

3. **运行项目**
   - 项目导入后自动编译
   - 在模拟器中查看效果
   - 或在真机上预览

## 功能使用指南

### 第一次使用

#### 1. 完善个人信息
首次使用需要先注册个人信息：

1. 点击底部 TabBar 的"我的"
2. 在欢迎页面填写个人信息：
   - **昵称**: 您的显示名称 (必填，最多20字符)
   - **微信号**: 用于其他用户联系您 (必填，最多30字符)
3. 点击"完成注册"

⚠️ **注意**: 微信号是必填项，其他用户会通过这个微信号添加您为好友。

### 浏览需求

#### 首页/论坛
1. 打开小程序，默认显示首页
2. 以卡片形式展示所有搭子需求
3. 每个卡片显示：
   - 活动类型标签 (看展/聚餐/运动/电影/其他)
   - 活动时间和地点
   - 简要描述
   - 费用说明
   - 发布者信息

#### 刷新列表
- **下拉刷新**: 在列表顶部下拉即可刷新
- **查看最新**: 新发布的需求会自动显示在列表顶部

### 查看详情

1. 点击首页的任意需求卡片
2. 进入详情页查看完整信息：
   - 活动类型
   - 活动标题
   - 活动地点
   - 活动时间
   - 详细描述
   - 费用说明
   - 发布者信息
3. 点击"获取微信号"按钮
4. 显示发布者的微信号
5. 点击"复制"按钮复制微信号
6. 到微信添加好友

### 发布需求

⚠️ **前提条件**: 必须先完善个人信息

1. 点击底部 TabBar 的"发布"
2. 填写需求信息：
   - **活动类型**: 从下拉列表选择 (看展/聚餐/运动/电影/其他)
   - **活动标题**: 简短描述您的需求 (最多50字符)
   - **活动地点**: 具体地点名称
   - **活动日期**: 选择日期
   - **活动时间**: 选择时间点
   - **详细描述**: 详细说明需求和期望 (最多500字符)
   - **费用说明**: 如"AA制"、"免费"、"人均100元"等
3. 所有字段均为必填项
4. 点击"发布需求"
5. 发布成功后自动跳转到首页

#### 表单验证
- 所有字段不能为空
- 标题最多50字符
- 描述最多500字符
- 活动日期不能早于今天

### 管理我的发布

#### 查看我的发布
1. 进入"我的"页面
2. 向下滚动查看"我的发布"列表
3. 显示您发布的所有需求

#### 编辑个人信息
1. 在"我的"页面点击"编辑"按钮
2. 修改昵称或微信号
3. 点击"保存"

#### 删除需求
1. 在"我的发布"列表中找到要删除的需求
2. 点击"删除"按钮
3. 确认删除操作
4. 删除后该需求从所有页面消失

## 页面导航

### TabBar (底部导航栏)
- **首页**: 浏览所有搭子需求
- **发布**: 发布新的搭子需求
- **我的**: 个人中心和我的发布

### 页面跳转
- 首页 → 详情页: 点击需求卡片
- 详情页 → 返回: 点击左上角返回按钮或系统返回
- 发布 → 首页: 发布成功后自动跳转
- 任意页面 → 个人中心: 点击 TabBar "我的"

## 数据说明

### 本地存储
项目使用微信小程序的本地存储 API 模拟数据持久化：

- **needs**: 存储所有需求数据
- **userInfo**: 存储当前用户信息

### Mock 数据
首次启动自动加载 3 条示例数据：
1. 周末一起看艺术展
2. 寻找火锅搭子
3. 一起打羽毛球

### 数据同步
- 发布新需求后立即显示在首页
- 删除需求后从所有位置移除
- 编辑个人信息后，已发布需求的用户信息自动更新

## 常见问题

### Q1: 为什么不能发布需求？
**A**: 必须先在"我的"页面完善个人信息（昵称和微信号）

### Q2: 如何联系发布者？
**A**: 
1. 进入需求详情页
2. 点击"获取微信号"
3. 复制微信号
4. 在微信中添加好友

### Q3: 可以编辑已发布的需求吗？
**A**: 当前版本暂不支持需求编辑，但可以删除后重新发布

### Q4: 数据会保存吗？
**A**: 数据保存在本地存储中，除非清除小程序数据或卸载小程序

### Q5: 可以查看历史需求吗？
**A**: 所有活跃的需求都显示在首页，可以随时浏览

### Q6: 如何刷新列表？
**A**: 在首页列表顶部下拉即可刷新

### Q7: 为什么看不到图标？
**A**: 项目使用了简单的占位图标，建议替换为专业设计的图标

## 注意事项

### 隐私保护
⚠️ **重要**: 微信号等联系方式会显示给所有用户，请谨慎填写

### 安全建议
1. 不要填写真实的敏感信息
2. 首次见面选择公共场所
3. 注意个人财物安全
4. 如遇可疑情况及时停止联系

### 使用建议
1. 填写清晰准确的需求描述
2. 明确费用说明，避免误解
3. 及时回复联系请求
4. 完成活动后可删除需求
