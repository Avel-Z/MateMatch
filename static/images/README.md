# Images Directory

This directory contains image assets for the MateMatch mini program.

## TabBar Icons
- home.png / home-active.png - Home page icons
- publish.png / publish-active.png - Publish page icons
- profile.png / profile-active.png - Profile page icons
- message.png / message-active.png - Message/Chat page icons

## Default Avatar
- avatar-default.png - Default user avatar

## Icon Specifications
- Size: 80x80 pixels
- Format: PNG with RGBA transparency
- Inactive state color: #999999 (gray)
- Active state color: #4CAF93 (theme color)
